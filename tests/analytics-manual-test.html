‚úÖ Manual Test Checklist

index.html Tests

  • Navigation Click Click any navigation link (About, Experience, Skills, etc.) in the header or mobile menu.
    Expected: navigation_click event with link_text, nav_type, destination_section
  • Experience Card Click Click on a company card in the Experience section.
    Expected: experience_interaction event with company, role, action
  • Portfolio Carousel Next Click the next button on the portfolio carousel.
    Expected: portfolio_interaction event with interaction_type: 'carousel_next', project_title
  • Portfolio Carousel Prev Click the previous button on the portfolio carousel.
    Expected: portfolio_interaction event with interaction_type: 'carousel_prev'
  • Portfolio Carousel Dot Click a dot navigation button on the carousel.
    Expected: portfolio_interaction event with interaction_type: 'dot_click'
  • Testimonials Next Click the next button on testimonials carousel.
    Expected: testimonials_interaction event with action: 'next', is_auto_rotate: false
  • Testimonials Hover Hover over a testimonial card (pause auto-rotate).
    Expected: testimonials_interaction event with action: 'pause_on_hover'
  • Contact - LinkedIn Click a LinkedIn link.
    Expected: contact_interaction event with contact_method: 'linkedin', location
  • Contact - Email Click an email link.
    Expected: contact_interaction event with contact_method: 'email'
  • Contact - GitHub Click a GitHub link.
    Expected: contact_interaction event with contact_method: 'github'
  • CTA Button Click "Show All Projects" or other CTA buttons.
    Expected: cta_click event with cta_text, destination
  • Back to Top Scroll down, then click the back to top button.
    Expected: back_to_top_click event with scroll_position, scroll_percentage
  • Mobile Menu Toggle On mobile (resize window < 480px), click hamburger menu.
    Expected: mobile_menu_toggle event with action: 'open', trigger_type: 'hamburger_click'

portfolio.html Tests

  • Filter Click Click a filter button (Business Intelligence, Data Visualization, etc.).
    Expected: filter_interaction event with filter_category, results_count, previous_filter
  • Filter All Click the "All" filter button.
    Expected: filter_interaction event with filter_category: 'all'

üîç Verify in Google Analytics

To see events in GA4 in real-time:
  1. Go to analytics.google.com
  2. Select your property: Rafi Atha Portfolio
  3. Click Reports ‚Üí Realtime
  4. Or use Admin ‚Üí DebugView for detailed event inspection
  5. Perform actions on your site (click buttons, navigate, etc.)
  6. Events should appear within a few seconds
‚ö†Ô∏è Important Notes:
  • Enhanced Measurement events (page_view, scroll, click) are tracked automatically by GA4
  • Custom events may take 24-48 hours to appear in standard reports (but show immediately in Realtime/DebugView)
  • If events don't appear, check browser console for errors
  • Make sure you're testing on localhost:8000 (dev server must be running)

‚å®Ô∏è Console Helper Commands

Paste these commands in the browser console for quick debugging:

// View all captured events
window.gtagEvents

// View last event
window.gtagEvents[window.gtagEvents.length - 1]

// Filter events by type
window.gtagEvents.filter(e => e.name === 'navigation_click')

// Count events by name
window.gtagEvents.reduce((acc, e) => {
    acc[e.name] = (acc[e.name] || 0) + 1;
    return acc;
}, {})

// Clear events log
window.gtagEvents = []

// Check if gtag is loaded
typeof window.gtag === 'function'

// Check if dataLayer exists
Array.isArray(window.dataLayer)
